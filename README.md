# Eco3s API Backend

## Description

this is the main backend service of [Eco3s](https://github.com/eco3s) project.

## Installation

```bash
$ git clone https://github.com/eco3s/backend
$ cd backend
$ pnpm install --frozen-lockfile
```

## Running

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```

## Test

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e

# test coverage
$ pnpm test:cov
```
